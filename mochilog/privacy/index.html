モチログ プライバシーポリシー

最終更新日: 2026年7月12日

「モチログ」（以下「本アプリ」）は、注入・美容施術の記録を残すためのアプリで、ユーザーのプライバシーを最優先に設計されています。本アプリは個人情報を収集せず、いかなるデータも外部サーバーに送信しません。アカウント登録やログインも不要です。

1. 収集する情報

本アプリは個人を特定できる情報を収集しません。施術の記録（製剤・部位・量・日付・クリニック名・費用・メモ）、チェックイン、経過写真、添付書類は、すべてお使いの端末内にのみ保存され、開発者や第三者がアクセスすることはありません。

2. 写真・書類について

アプリ内カメラで撮影した経過写真や、追加した書類の画像は、本アプリ専用の保護された領域（端末内）にのみ保存され、写真ライブラリ（カメラロール）には書き込みません。PDFなどで書き出す際には、位置情報（GPS）や撮影機材などのExif情報を除去してから出力します。

3. Face ID / Touch ID について

アプリロック機能で Face ID / Touch ID を使う場合、これはロックの解除のためだけに使用します。顔・指紋などの生体データは Apple の Secure Enclave 内で処理され端末の外に出ることはなく、本アプリは認証の成否のみを受け取ります。本アプリが生体データを取得・保存することはありません。

4. 課金について

Premium（買い切り）の購入は Apple の App Store（StoreKit）を通じて処理されます。お支払い情報は Apple が管理し、本アプリが取得・保存することはありません。月額・年額などの継続課金はありません。

5. 通知

経過チェックのリマインドなどの通知は、すべて端末内で生成されるローカル通知です。通知の内容が外部に送信されることはありません。既定では、プライバシーに配慮して通知本文に施術名・部位を表示しません（設定で変更できます）。

6. 第三者・トラッキング

本アプリは広告SDK・解析SDK・トラッキング技術を一切使用していません。ユーザーをトラッキングせず、IDFA等の識別子も取得しません。外部との通信を行いません。

7. データの保存・移行・削除

本アプリはクラウド同期を行わないため、データはお使いの端末内にのみ保存されます。機種変更や端末の初期化に備え、「設定 > データの書き出し（PDF）」でのバックアップをおすすめします。アプリ内の「設定 > すべてのデータを削除」から全データを削除でき、端末から本アプリを削除するとすべてのデータも端末から削除されます。
