# app-legal

iOS アプリ向けのプライバシーポリシー・利用規約等の公開リポジトリ。

GitHub Pages として公開されます。

## 構成

- `tarotai/privacy/` — タロットAI プライバシーポリシー
- `seionbiyori/` — 静音日和 法務・サポート入口
- `seionbiyori/privacy/` — 静音日和 プライバシーポリシー
- `seionbiyori/terms/` — 静音日和 利用規約
- `seionbiyori/support/` — 静音日和 サポート

## 編集方法

各 `index.html` を直接編集 → push すると自動で GitHub Pages に反映される。
