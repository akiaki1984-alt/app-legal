プライバシーポリシー

最終更新日: 2026年6月27日

RealRep（以下「本アプリ」）は、スクワットの記録をフレンドと共有するソーシャル・フィットネスアプリです。本ポリシーは、本アプリが取得・利用する情報と、その取り扱いについて説明します。

取得する情報

  • アカウント情報: Appleでサインインを利用して作成されるアカウント識別子（ユーザーID）、表示名、ユーザー名。Appleの設定によりメールアドレスは非公開にできます。
  • あなたが投稿するコンテンツ: スクワット時に撮影し、あなたが投稿を選んだ写真、キャプション、コメント、リアクション、レップ数や運動時間などの記録。
  • フレンド・交流情報: フレンド関係、送受信した「挑戦状」、いいね。
  • デバイス・利用情報: 不具合解析や品質改善のための一般的な利用ログ、プッシュ通知トークン。

カメラとモーションセンサーは、スクワットの計測と「証明写真」の作成のために使用します。撮影中の映像は保存されず、あなたが選んだ1枚の写真のみが投稿に使われます。

情報の利用目的

  • 記録の保存・表示、フィードやフレンド機能などのサービス提供のため
  • プッシュ通知の配信のため
  • 不具合の解析、機能改善、安全なコミュニティの維持（通報対応など）のため

第三者サービス

本アプリはバックエンドに Google の Firebase（Authentication / Cloud Firestore）を利用しています。アカウント情報・投稿・コメント等は、日本（asia-northeast1）リージョンの Firestore に保存されます。Firebase の取り扱いについては Google のプライバシーポリシーをご確認ください。

本アプリは、あなたの個人情報を販売しません。法令に基づく場合を除き、第三者に提供しません。

公開範囲

あなたが投稿したコンテンツは、選択した公開範囲（フレンド／公開）に従って他のユーザーに表示されます。非公開アカウントに設定すると、投稿が公開フィードに表示されなくなります。

データの削除

アプリ内の「設定 → 法的情報 → アカウントを削除」から、いつでもアカウントと関連データ（プロフィール、記録、投稿）の削除を申請できます。削除は元に戻せません。

子どものプライバシー

本アプリは13歳未満の利用を想定していません。
