人相診断AI プライバシーポリシー

最終更新日: 2026年5月30日

本アプリ「人相診断AI」（以下「本アプリ」）は、ユーザーのプライバシーを最優先に設計されています。本ポリシーは、本アプリが取り扱う情報について説明します。

1. 収集する情報 本アプリは、個人を特定する情報をサーバーへ送信・収集しません。

・顔写真について 　ユーザーが撮影・選択した顔写真は、お使いの端末（iPhone/iPad）の中だけで Apple の Vision フレームワークにより解析されます。写真そのものや画像データが外部に送信・保存されることは一切ありません。解析後、端末内に保存されるのは「診断タイプ」と「4つの特徴量（数値）」のみで、元の写真は保存されません。

・診断履歴について 　診断結果・相性結果は端末内のデータベース（SwiftData）にのみ保存されます。これらはクラウドに送信されません。アプリを削除すると履歴も削除されます。

2. オンデバイスAIについて 解釈文の生成には Apple Intelligence（オンデバイスAI）を使用します。処理は端末内で完結し、入力内容が外部に送信されることはありません。

3. 購入情報について Pro プラン等の購入は Apple の App Store / StoreKit を通じて処理されます。課金情報は Apple が管理し、本アプリの開発者がクレジットカード情報等を取得することはありません。

4. 第三者への提供 本アプリは、ユーザー情報を第三者に販売・提供しません。広告 SDK や解析 SDK は使用していません。

5. お子様のプライバシー 本アプリは特定の年齢層を対象としていませんが、お子様が利用する場合は保護者の管理のもとでご利用ください。

6. 本ポリシーの変更 本ポリシーは予告なく変更されることがあります。重要な変更がある場合はアプリの更新を通じてお知らせします。
